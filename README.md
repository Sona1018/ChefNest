# 🍽️ ChefNest

**ChefNest** is a full-stack MERN web application that connects users with professional chefs for home dining, events, and customized cooking experiences. The platform allows users to explore chefs, make bookings, and manage requests, while administrators can efficiently manage chefs, bookings, users, and notifications through a dedicated dashboard.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sona1018/ChefNest.git
```

```
cd ChefNest
```

---

## Backend

```
cd ChefNestBackend
npm install
npm start
```

---

## Frontend

```
cd chefNestFrontend
npm install
npm start
```

---

## Dashboard

```
cd chefnestdashboard
npm install
npm start
```

---

## Environment Variables

### Backend (.env)

```env
PORT=8000

MONGO_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET

CLOUDINARY_NAME=YOUR_NAME

CLOUDINARY_API_KEY=YOUR_KEY

CLOUDINARY_API_SECRET=YOUR_SECRET
```

### Frontend

```env
REACT_APP_API_URL=https://chefnest-backend.onrender.com
```

### Dashboard

```env
REACT_APP_API_URL=https://chefnest-backend.onrender.com
```
